# Pro01

A Laravel-based web application.

## Requirements
* PHP >= 8.2
* Composer
* Node.js & NPM
* MySQL or SQLite

## Installation
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Seavmoeurng/Pro01.git](https://github.com/Seavmoeurng/Pro01.git)
   cd Pro01

2.Setup:

    Run composer install

    Run cp .env.example .env

    Run php artisan key:generate

    Run php artisan migrate
